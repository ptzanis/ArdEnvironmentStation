Introduction

An overparameterized file is a file (panel, script, library, etc) that exists in multiple paths of a project. The problem caused by these files is that, in many cases, users are not aware of the file that will be loaded by WinCC OA at run time. In other words, you should avoid having overparameterized files in your project.

A hash error measns that the MD5 hash of a file in the installed components folder is different than the the MD5 hash of the same file situated in the directory from which the component was installed. In other words the file has ben modified.
